群规则

规范原则：

注意：违反原则者一律拉黑踢出无需理由

（一）攻击我国政治制度、法律制度的内容

  1. 调侃、讽刺、反对、谩骂中国特色社会主义道路、理论、制度和文化以及国家既定重大方针政策的；

  2. 对宪法等国家重大法律法规的制定、修订进行曲解、否定、攻击、谩骂，或对其中具体条款进行调侃、讽刺、反对、谩骂的；

  3. 削弱、背离、攻击、诋毁中国共产党领导的；

  4. 对改革开放以来国家所实行的重大方针政策进行调侃、否定、攻击的；

  5. 篡改、娱乐化解读我国政治制度和法律制度中的特定名词称谓的；

（二）分裂国家的内容

  1. 反对、攻击、曲解“一个中国”“一国两制”的；

  2. 反映台独、港独、藏独、疆独等的言行、活动、标识的，包括影像资料、作品、语音、言论、图片、文字、反动旗帜、标语口号等各种形式（转播中央新闻单位新闻报道除外）；

  3. 持有台独、港独、藏独、疆独等分裂国家立场的艺人及组织团体制作或参与制作的节目、娱乐报道、作品宣传的；

  4. 对涉及领土和历史事件的描写不符合国家定论的；

（三）损害国家形象的内容

  1. 贬损、玷污、恶搞中国国家和民族的形象、精神和气质的；

  2. 以焚烧、毁损、涂划、玷污、践踏、恶搞等方式侮辱国旗、国徽的；在不适宜的娱乐商业活动等场合使用国旗、国徽的；

  3. 篡改、恶搞国歌的；在不适宜的商业和娱乐活动中使用国歌，或在不恰当的情境唱奏国歌，有损国歌尊严的；

  4. 截取党和国家领导人讲话片段可能使原意扭曲或使人产生歧义，或通过截取视频片段、专门制作拼凑动图等方式，歪曲放大展示党和国家领导人语气语意语态的；

  5. 未经国家授权或批准，特型演员和普通群众通过装扮、模仿党和国家领导人形象，参加包括主持、表演、演讲、摆拍等活动，谋取利益或哗众取宠产生不良影响的 （依法批准的影视作品或文艺表演等除外）；

  6. 节目中人物穿着印有党和国家领导人头像的服装鞋帽，通过抖动、折叠印有头像的服装鞋帽形成怪异表情的；

（四）损害革命领袖、英雄烈士形象的内容

  1. 歪曲、丑化、亵渎、否定革命领袖、英雄烈士事迹和精神的；

  2. 不当使用及恶搞革命领袖、英雄烈士姓名、肖像的；

（五）泄露国家秘密的内容

  1. 泄露国家各级党政机关未公开的文件、讲话的；

  2. 泄露国家各级党政机关未公开的专项工作内容、程序与工作部署的；

  3. 泄露国防、科技、军工等国家秘密的；

  4. 私自发布有关党和国家领导人的个人工作与生活信息、党和国家领导人家庭成员信息的；

（六）破坏社会稳定的内容

  1. 影响公共秩序与公共安全的群体性事件的；

  2. 传播非省级以上新闻单位发布的灾难事故信息的；

  3. 非新闻单位制作的关于灾难事故的影响、后果的节目的；

（七）损害民族与地域团结的内容

  1. 通过语言、称呼、装扮、图片、音乐等方式嘲笑、调侃、伤害民族和地域感情、破坏安定团结的；

  2. 将正常的安全保卫措施渲染成民族偏见与对立的；

  3. 传播可能引发误解的内容的；

  4. 对独特的民族习俗和宗教信仰猎奇渲染，甚至丑化侮辱的；

  5. 以赞同、歌颂的态度表现历史上民族间征伐的残酷血腥战事的；

（八）违背国家宗教政策的内容

  1. 展示宗教极端主义和邪教组织及其主要成员的活动，以及他们的“教义”与思想的；

  2. 不恰当地比较不同宗教、教派的优劣，可能引发宗教、教派之间矛盾和冲突的；

  3. 过度展示和宣扬宗教教义、教规、仪式内容的；

  4. 将宗教极端主义与合法宗教活动混为一谈，将正常的宗教信仰与宗教活动渲染成极端思想与行动，或将极端思想与行动解释成正常的宗教信仰与宗教活动的；

  5. 戏说和调侃宗教内容，以及各类恶意伤害民族宗教感情言论的；

（九）传播恐怖主义的内容

  1. 表现境内外恐怖主义组织的；

  2. 详细展示恐怖主义行为的；

  3. 传播恐怖主义及其主张的；

  4. 传播有目的、有计划、有组织通过自焚、人体炸弹、打砸抢烧等手段发动的暴力恐怖袭击活动视频（中央新闻媒体公开报道的除外），或转发对这些活动进行歪曲事实真相的片面报道和视频片段的；

（十）歪曲贬低民族优秀文化传统的内容

  1. 篡改名著、歪曲原著精神实质的；

  2. 颠覆经典名著中重要人物人设的；

  3. 违背基本历史定论，任意曲解历史的；

  4. 对历史尤其是革命历史进行恶搞或过度娱乐化表现的;

（十一）恶意中伤或损害人民军队、国安、警察、行政、司法等国家公务人员形象和共产党党员形象的内容

  1. 恶意截取执法人员执法工作过程片段，将执法人员正常执法营造成暴力执法效果的；

  2. 传播未经证实的穿着军装人员打架斗殴、集会、游行、抗议、上访的；

  3. 正面展现解放军形象时用语过度夸张的；

（十二）美化反面和负面人物形象的内容

  1. 为包括吸毒嫖娼在内的各类违法犯罪人员及黑恶势力人物提供宣传平台，着重展示其积极一面的；

  2. 对已定性的负面人物歌功颂德的；

（十三）宣扬封建迷信，违背科学精神的内容

  1. 开设跳大神、破太岁、巫蛊术、扎小人、道场作法频道、版块、个人主页，宣扬巫术作法等封建迷信思想的；

  2. 鼓吹通过法术改变人的命运的；

  3. 借民间经典传说宣扬封建迷信思想的；

（十四）宣扬不良、消极颓废的人生观、世界观和价值观的内容

  1. 宣扬拜金主义和享乐主义的；

  2. 展示违背伦理道德的糜烂生活的;

  3. 宣传和宣扬丧文化、自杀游戏的；

  4. 展现同情、支持婚外情、一夜情的；

（十五）渲染暴力血腥、展示丑恶行为和惊悚情景的内容

  1. 表现黑恶势力群殴械斗、凶杀、暴力催债、招募打手、雇凶杀人等猖狂行为的；

  2. 细致展示凶暴、残酷的犯罪过程及肉体、精神虐待的；

  3. 细致展示吸毒后极度亢奋的生理状态、扭曲的表情，展示容易引发模仿的各类吸毒工具与吸毒方式的；

  4. 细致展示酗酒后失控状态的；

  5. 细致展示老虎机、推币机、打鱼机、上分器、作弊器等赌博器具，以及千术、反千术等赌博技巧与行为的；

  6. 展现过度的生理痛苦、精神歇斯底里，对普通观看者可能造成强烈感官和精神刺激，从而引发身心惊恐、焦虑、厌恶、恶心等不适感的画面、台词、音乐及音效的；

  7. 宣扬以暴制暴，宣扬极端的复仇心理和行为的;

（十六）展示淫秽色情，渲染庸俗低级趣味，宣扬不健康和非主流的婚恋观的内容

  1. 具体展示卖淫、嫖娼、淫乱、强奸等情节的；

  2. 直接展示性行为的；

  3. 视频中出现色情推广的；

  4. 展示呻吟、叫床等声音、特效的；

  5. 以猎奇宣扬的方式对境外“红灯区”进行拍摄的；

  6. 展现有性交易内容的夜店、洗浴按摩场所的；

  7. 表现和展示非正常的性关系、性行为的；

  8. 展示和宣扬不健康的婚恋观和婚恋状态的；

  9. 宣扬和炒作非主流婚恋观的；

  10. 以单纯感官刺激为目的，集中细致展现接吻、爱抚、淋浴及类似的与性行为有关的间接表现或暗示的， 包括裸露或长时间聚焦胸部、臀部等部位，聚焦走光、偷拍、凸点，渲染恋足、原味丝袜等性癖好；

  11. 有明显的性挑逗、性骚扰、性侮辱或类似效果的画面、台词、音乐及音效的；

  12. 展示男女性器官，或仅用肢体掩盖或用很小的遮盖物掩盖人体隐秘部位及衣着过分暴露的；

  13. 使用粗俗语言，展示恶俗行为的；

  14. 以隐晦、低俗的语言表达使人产生性行为和性器官联想的内容的；

  15. 以成人电影、情色电影、三级片被审核删减内容的影视剧的“完整版”“未删减版”“未删节版”“被删片段”“汇集版”作为视频节目标题、分类或宣传推广的；

  16. 以偷拍、走光、露点及各种挑逗性文字或图片作为视频节目标题、分类或宣传推广的；

  17. 使用易引发性联想的文字作为标题的；

（十七）侮辱、诽谤、贬损、恶搞他人的内容

  1. 侮辱、诽谤、贬损、恶搞历史人物及其他真实人物的形象、名誉的；

  2. 贬损、恶搞他国国家领导人，可能引发国际纠纷或造成不良国际影响的；

  3. 侮辱、贬损他人的职业身份、社会地位、身体特征、健康状况的；

（十八）有悖于社会公德的内容

  1. 以恶搞方式描绘重大自然灾害、意外事故、恐怖事件、战争等灾难场面的；

  2. 以肯定、赞许的基调或引入模仿的方式表现打架斗殴、羞辱他人、污言秽语的；

  3. 为违背公序良俗或游走在社会道德边缘的行为提供展示空间的；

（十九）不利于未成年人健康成长的内容

  1. 表现未成年人早恋的，以及抽烟酗酒、打架斗殴、滥用毒品等不良行为的；

  2. 人物造型过分夸张怪异，对未成年人有不良影响的；

  3. 展示未成年人或者未成年人形象的动画、动漫人物的性行为，或让人产生性妄想的；

  4. 侵害未成年人合法权益或者损害未成年人身心健康的;

（二十）宣扬、美化历史上侵略战争和殖民史的内容

  1. 宣扬法西斯主义、极端民族主义、种族主义的；

  2. 是非不分，立场错位，无视或忽略侵略战争中非正义一方的侵略行为，反而突出表现正义一方的某些错误的；

  3. 使用带有殖民主义色彩的词汇、称谓、画面的；

（二十一）其他违反国家有关规定、社会道德规范的内容

  1. 将政治内容、经典文化、严肃历史文化进行过度娱乐化展示解读，消解主流价值的；

  2. 从事反华、反党、分裂、恐怖活动的特定组织或个人制作或参与制作的节目，及其开设的频道、版块、主页、账号的；

  3. 违法犯罪、丑闻劣迹者制作或参与制作的节目的；

  4. 侵犯个人隐私，恶意曝光他人身体与疾病、私人住宅、婚姻关系、私人空间、私人活动的；

  5. 对国家有关规定已明确的标识、呼号、称谓、用语进行滥用、错用的；

  6. 破坏生态环境，虐待动物，捕杀、食用国家保护类动物的；

  7. 展示个人持有具有杀伤力的危险管制物品的；

  8. 在节目中植入非法、违规产品和服务信息的；

100.其他有违法律、法规和社会公序良俗的。

规则本质

一.禁不良违法信息发布【四禁】

1.严禁发布违禁品、毒品、管制刀具、赌博、假证假票据、售卖考试答案、售卖个人信息相关的违法信息。

2.严禁发布政治敏感话题、封建迷信、淫秽、色情、暴力、凶杀、恐怖、教唆犯罪等相关的不良信息。不得口吐芬芳，爆粗口。不得人身攻击，人肉搜索。

3.严禁在群内发布病毒网址，恐吓网址。

4.严禁在群内造谣传谣。

5.严禁发布与国家的法律法规、制度政策相抵触的言论。

6.严禁在群内进行任何诈骗行为，本群对此保持零容忍的态度，一经发现将会对违规成员进行踢出、举报、公示。

7.严禁冒充任何群管理员、审核员。

8.严禁售卖电子烟等不利于青少年健康的物品。

9.严禁在本群宣传有关改装电动车、超速驾驶等违法内容。

二.禁不良黑恶势力入群

1.严禁任何黑恶势力进入本群，任何通过本群进行一切违法行为（包括但不限于打架斗殴、校园欺凌、寻衅滋事等）一经发现，将会上诉到国家信访办以及拨打110和12345市长热线。坚决打压群内打架、约架等不良风气。

2.严禁肆意骚扰他人（包括但不限于私下言语攻击，人肉搜索，言语威胁等），须有文明举止，树正气之风。本群对此保持零容忍的态度，一经发现将会对违规成员进行踢出、举报、公示。

3.严禁在本群宣扬（扛把子、靠山）主义，须有道德修养，有自我约束，遵纪守法。

三.禁色情低俗信息发布

1.严禁发布色情交易、不正当交友信息。

2.严禁发布直接或隐晦表现人体性部位，暗示性行为的图片及内容。

3.严禁发布带有侵犯个人隐私性质的偷拍走光漏点人体部位的图片。

4.严禁发布表示对未满14周岁儿童产生性幻想、性冲动的内容。须严格自律，注重言行。

四.禁不友善攻击言论发布

1.严禁以不文明语言对他人进行攻击、谩骂，或者进行负面评价。

2.严禁攻击、抹黑与自身文化不同的地域、民族、国家。

3.严禁不以正常交流为目的，通过恶意反复@、回复、私信等方式骚扰他人。

4.严禁以不友好的方式激怒他人，迫使对方对自己的言论做出回应，故意挑起争端。

5.严禁歧视老、弱、病、残、孕者；同性恋、泛性恋、双性恋者；与自己种族、地区、家境等不同者。拒绝炫耀与攀比。

6.不得极端追星/黑星，须自觉树立正确的价值观。

五.应文明友善用语【一应】

1.工作交流不掺杂负面情绪，坚持积极向上、文明用语，减少发布未经公开报道，未经证实的小道消息。

2.应保持群聊之间的友好、和谐。

3.应维护学校，尊敬师长，热爱学习。

六.群管团队温馨提醒【一提醒】

1.不要涉及金钱交易及产生利益关系。

2.不要随意公开任何与自身相关的信息与地址，防止网络暴力及骚扰。

3.不要讨论带有争议性的明星、名人等，以免引起争论。

4.不要与陌生人随意线下见面。

5.群成员若有个人无法调节的问题，可以私信群管团队。

详细处罚规定

1.刷屏：连续发送3条或5行及以上无意义或重复的文字消息、连续发送3条及以上图片或视频消息，违者禁言10分钟至12小时。

2.传播负能量：在群内发送负能量消息或刻意引导其他成员讨论相关话题（如：自杀、抑郁症等相关内容），违者禁言1至12小时。

3.人身攻击：对于任何一名群成员进行言语辱骂、针对性语言攻击，违者禁言1至12小时。

4.引战：蓄意发送挑起其他群成员矛盾或令其他群成员强烈不满的消息，违者禁言1至24小时。

5.恶搞：未经允许发送包含他人头像、名片、聊天记录的消息、未经允许恶意篡改他人头像、群名片，并将修改后的内容发送至群内，违者禁言1至6小时。

6.传播谣言：对未经证实的新闻内容妄下定论，在群内盲目传播谣言；对本群消息断章取义并发布至其他群聊或平台，违者禁言12小时及以上，严重者踢出群聊。

7.违法内容：在群内发送任何形式的包含黄色、赌博、毒品、血腥暴力等违反中华人民共和国相关法律的消息（包括闪照），违者踢出群聊。

8.频繁加群：同一账号在7天内退、加群操作超过3次及以上，违者踢出群聊并不再接受加群申请。

9.侵犯个人隐私：在群内发送或胁迫他人发送个人照片、手机号、家庭住址等个人信息，违者禁言5天及以上，严重者踢出群聊并全网通报。

10.虚假广告：在群内发送虚假宣传的福利广告（如：拼多多砍价、QQ会员免费领取等），违者禁言1至24小时。

11.政治敏感：在群内讨论或引导其他群成员讨论国家领导人、国家疆域矛盾等敏感话题，发送错误的中国地图、抹黑国旗或人民币等违法违规的图片消息，违者踢出群聊。

12.骚扰：通过本群私聊其他群成员进行骚扰、辱骂、消息轰炸等行为，违者踢出群聊并全网通报。

13.网络代理：在群内发送“VPN”相关词汇，违者禁言30分钟至6小时并撤回所有相关消息。

14.红包：发送总额小于0.5元的红包、使用任何作弊手段抢红包、发送无解的接龙红包或将他人的接龙红包引导为无解红包，违者禁言1小时及以上，严重者踢出群聊。

15.诋毁：在任何群聊或平台发布恶意诋毁本群的言论，违者踢出群聊并全网通报。

16.低俗：发送低俗、烂梗内容消息，违者禁言1至24小时。

17.漏洞代码：通过任何手段发送利用QQ漏洞达到@全体成员的消息，违者踢出群聊。

18.网站链接：发送钓鱼、诈骗、黄色网站的链接消息，违者禁言1至10天，严重者踢出群聊。

19.交流语气：使用阴阳怪气等可能令他人感到不适的语气进行交流（包括文字、图片、表情消息），违者禁言30分钟至6小时。

20.人肉搜索：通过任何手段获取任何人的家庭住址、手机号、身份证号、账号密码等个人敏感隐私并将其发送至群内或其他群聊或平台，违者踢出群聊并全网通报，严重者将被管理委员会报案，并依照《中华人民共和国民法典》相关法律进行处理。

21.性别话题：在群内发送或引导他人讨论与性别相关的话题，违者禁言6小时至3天。

22.攀比：刻意发送带有攀比意义（如：学习成绩等）可能引起他人不适的消息，违者禁言1至12小时。

23.敏感词：在群内发送真实存在的地名（包括但不限于国家、省、市等）请用首字母或其他形式代替

24.管理员警告：对于管理员警告不予理会，违者按相关规定进行处理。

25.以上群规屡次违反者或严重者都将被踢出群聊并全网通报。

管理员规则

规则

新管理员制度提倡“积极投票”“积极参与”“积极保障与管理”三大要素，并不向以往一样直接依靠关系申请就会获得。另外，不要试图不劳而获，所有的群管理员都将经过审核后才能上任。

1.管理员应参与本群的日常维护和管理监督工作，及时做好群成员的加入、移出等工作。

2.管理员不定时进行一次轮换，如旧管理员仍想继续上任，则群主来决定是否能有资格继续连任。如管理员中途有违规超过两次或退群/移除群聊，则永久不可获得管理员资格。

3.管理员应有积极的心态，对群员友好，管理得当，适当保持活跃，管理员不允许不按照“驻马店市第十中学校园群使用规则”来私自给群员进行额外处分，违者该账号永久不能获得管理员资格。

4.如若管理员发现他人发言掺有扛把子主义，打架约架等字样或是骚扰群成员，应当立即禁言并撤回违规内容。

最低申请标准

1.有良好信用，不超过2次的违规，活跃性一般。

2.语言稍粗俗，有擦边情况，不骚扰群员。

3.不得发广告，群组宣传。

4.不允许超过1周不活跃，否则资格取消。

5.不得漠视群规，管理员可在保证不违规群规的前提下，用自己的方式来管理群组，本群是提倡的。

6.至少加入本群一个月以上。

疫情防控规则

规则

1.面对疫情，应保持理智的态度面对。凡是拿疫情开玩笑的， 不把疫情当回事的，贸然践踏疫情防控规定的， 一律踢出群外。

2.凡是散播疫情谣言的，传播小道消息的（任何未经官方证实的流传的信息都为小道消息 ）一律踢出群外。

3.面对疫情，不必过于慌张，严格遵守疫情防控规定，及时参与全民核酸检测。

4.不要把自己的理论消息当作官方来看待和传播。

5.及时观察驿城区疫情防控办的消息并严格遵守。
